Introduction

The Menstrual Awareness Project is an educational initiative aimed at spreading awareness about menstrual health, hygiene, and related health issues. This project includes interactive quizzes and informative workshop listings to educate both men and women about menstrual well-being and break societal taboos.

Features

1. Awareness Quiz

Interactive quiz to educate users about menstrual health.

Multiple-choice questions with instant feedback.

Score tracking to encourage learning.

Encouraging messages for participants.

2. Menstrual Awareness Workshops

A list of upcoming workshops focused on menstrual health.

Details include workshop title, date, location, and description.

Helps users find and attend relevant sessions on menstrual hygiene, PCOS, and nutrition.

Technologies Used

Next.js (React framework for building UI components)

Tailwind CSS (for styling and responsive design)

Framer Motion (for animations and UI transitions)

Installation & Setup

To run this project locally, follow these steps:

Prerequisites

npm or yarn package manager
